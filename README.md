## About Splity API
Splitly API is an API made for [Splitly](https://github.com/philip0014/splitly-ui) web application.

## Features
<ul>
	<li>Authentication & Google sign in</li>
  <li>JWT token</li>
  <li>A lot of CRUD to MongoDB</li>
  <li>Reactive service with RxJava</li>
</ul>
